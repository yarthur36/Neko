Ceci est le manuel d'utilisation du jeu Neko.

1 ) Lancer le jeu :

	Pour lancer le jeu, assurez vous que tout les fichiers .py et .txt initialement présents dans le dossier Neko y soit toujours. Ensuite, il vous suffira d'éxécuter le fichier "Main.py" sous python3 dans un terminal Linux.


2 ) Objectif du jeu : 

	Le but du jeu Neko est d'enquêter sur un meurtre. Il vous faudra donc découvrir les différents lieux qui vous seront proposés et intéragir avec les différents personnages présent ( les personnages ont la forme de 8 ).
	De plus, il faudra terminer plusieurs labyrinthes pour parvenir en ces lieux.( Attention, toute collision avec les murs du labyrinthe vous rameneront au point de départ de celui-ci).
	
3 ) Se déplacer : 

	Il y a au total 4 touches de déplacement proposées dans le jeu Neko.
	La touche 'Z' vous permet de vous déplacer vers le haut.
	La touche 'Q' vous permet de vous déplacer vera la gauche.
	La touche 'S' vous permet de vous déplacer vers le bas.
	La touche 'D' vous permet de vous déplacer vers la droite.

4 ) Des options :

	Dès le début du jeu, une option vous sera proposer : choisir la forme de votre inspecteur. 
	Vous pourrez donc appuyer sur la touche 'O' ou '8' de votre clavier pour choisir le personnage correspondant.
	Une fois en jeu, vous aurez la possibilité de changer la couleur de votre inspecteur autant que vous le voulez et quand vous le voulez. Pour cela appuyez sur la touche 'c'
	Vous aurez également la possibilité de changer la couleur de la carte en appuyant sur la touche 'v'

	De plus, pour faciliter l'évaluation de ce projet, il vous est possible de mettre et d'enlever les retour à la position initiale dans les labytinthes en ca de collisions. Pour cela, appuyer sur la touche 'p'.
	


5 ) Quitter le jeu :

	Pour quitter le jeu, il vous suffira d'appuyer sur la touche 'échap'.
